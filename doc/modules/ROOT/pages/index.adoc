= Boost.RuntimeServices

Boost.RuntimeServices is a portable, low-level C++ library which provides
facilities for creating and accessing optional services during runtime.

* Require only C++11
* Works without exceptions
* Works without RTTI
* Fast compilation, few templates

== Embedded

Boost.Runtime_services works great on embedded devices.
It is designed to work without exceptions and RTTI if desired.

== Tested Compilers

Boost.Runtime_services has been tested with the following compilers:

* clang: 3.8, 4, 5, 6, 7, 8, 9, 10, 11, 12
* gcc: 4.8, 4.9, 5, 6, 7, 8, 9, 10, 11
* msvc: 14.1, 14.2, 14.3
